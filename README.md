Totk-Cooking-simulator
====

## 简介

一个简单的Python王国之泪料理模拟器或者说计算器。

## 示例

```python
from simulator import TotkCookSim

sim = TotkCookSim()
result = sim.cook(['天空蘑菇'], area_lang='CNzh')
print(result)
result = sim.cook(['Apple', 'Apple','Apple','Apple','Apple'])
print(result)
```
